# Floor Plans

Drop files into the relevant subfolder. No code changes needed.

## File structure

```
4-bed-semi-detached/   (The Semi-Detached — Plot 7258)
  ground.jpg + first.jpg + floor-plan.pdf

4-bed-terrace/         (The Terrace — Plot 7253)
  ground.jpg + first.jpg + floor-plan.pdf

6-bed-terrace/         (The Six — Plot 7270, 3 floors)
  ground.jpg + first.jpg + second.jpg + floor-plan.pdf

5-bed-semi-detached/   (The Five — Plot 7261)
  ground.jpg + first.jpg + floor-plan.pdf

4-bed-block-flats/     (The Pavilion — Plot 7266, 1 floor)
  ground.jpg + floor-plan.pdf
```

## Notes
- Images: JPG/PNG, minimum 1200px wide
- PDF: one per property, all floors in one file
